# Laptop Price Prediction

Predicting laptop prices using a custom-built Linear Regression model with full statistical analysis.

## What's Inside
- Full EDA with correlation heatmaps and outlier detection (IQR)
- Custom LinearRegressionStatisticalView class built from scratch
- Statistical inference: beta coefficients, F-statistics, confidence intervals, R²
- Preprocessing: log transformation, one-hot encoding, standard scaling

## Tools
Python, pandas, NumPy, SciPy, Matplotlib, Seaborn, Plotly
